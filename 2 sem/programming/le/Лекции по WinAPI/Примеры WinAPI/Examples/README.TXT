Этот файл содержит краткое описание примеров
приложений из первого тома книги 
Мешкова А.В. и Тихомирова Ю.В.
"Visual C++ и MFC", полные тексты программ 
этих примеров приведены на данной дискете 
в одноименных каталогах.

1. Использование примеров:

   - каталоги с примерами можно переписать на локальный или сетевой
     диск любыми средствами как MS-DOS, так и Windows - все 
     файлы не имеют длинных имен;

   - для использования примеров, а также их модификации или
     перекомпиляции требуется Visual С++ версии не ниже 4.0;

2. Список примеров приложений:

   - APIShell - пример из главы 4 (APISHELL)
     применение функций Win32 API 
     для создания приложений, использующих 
     область уведомления панели задач
         
   - APISync - пример из главы 5 (APISYNC)
     приложение, использующее синхронизирующие объекты для
     обнаружения копии приложения в памяти и управления 
     выделением ресурсов

   - APIWin - пример из главы 6 (APIWIN)
     текстовый редактор, созданный на основе функций Win32 API

   - ControlBars - пример из 13-ой главы (CTRLBARS)
     создание и использование панелей элементов управления

   - DlgSmpl - пример из главы 14 (DLGSMPL)
     приложение, главным окном которого является блок диалога
     
   - First - пример из главы 9 (FIRST)
     минимальное приложение на базе классов библиотеки MFC

   - MDI - пример из главы 12 (MDI)
     многодокументное приложение

   - Styles - пример из главы 11 (STYLES)
     использование различных стилей при создании окон

   - Template - пример из главы 14 (TEMPLATE)
     создание блока диалога непосредственно в памяти, 
     т.е. без использования ресурсов
